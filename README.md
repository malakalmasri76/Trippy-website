A responsive travel website built with **React.js** that showcases popular destinations and tours.

## Features
- Dynamic display of travel destinations
- Responsive layout for desktop, tablet, and mobile
- Smooth animations and interactive UI elements

## Technologies
- React.js
- CSS / Flexbox / Grid
- Responsive Design

## Installation
1. Clone the repo: `git clone https://github.com/malakalmasri76/Trippy-App.git`
2. Install dependencies: `npm install`
3. Run the app: `npm start`
4. Open in browser: `http://localhost:3000`

## Challenges
- Implementing responsive design across multiple screen sizes
- Managing state for dynamic destination cards
